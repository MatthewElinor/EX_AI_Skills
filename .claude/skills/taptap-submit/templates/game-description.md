# TapTap Game Description Template

游戏描述模板，用于上架TapTap平台。

## 模板结构

### 标准模板

```
【游戏名称】是一款【类型】游戏。

【核心玩法描述】（一句话概括游戏的核心玩法）

【特色亮点】（列出3-5个游戏特色）

适合喜欢【相关类型/题材】的玩家，快来体验吧！
```

### 字数要求

- 简短描述：50-100字（用于列表展示）
- 完整描述：200-500字（用于详情页）

## 示例

### 示例1：类《杀戮尖塔》卡牌游戏

**简短描述**（80字）：
```
《卡牌冒险》是一款Roguelike卡牌战斗游戏。收集卡牌和遗物，在随机地图中挑战敌人，每次冒险都是全新体验！
```

**完整描述**（300字）：
```
《卡牌冒险》是一款Roguelike卡牌战斗游戏。

玩家将扮演冒险者，通过收集和升级卡牌、获取神秘遗物，在随机生成的地图中挑战各种敌人。每次冒险都是全新的体验，不同的卡牌组合带来无限策略可能。

核心特色：
- 丰富的卡牌系统，支持多种流派玩法
- 遗物系统提供永久增益，增强策略深度
- 随机地图与事件，每次冒险都不一样
- 精美的扁平卡通美术风格
- 上手简单，策略深度高

适合喜欢策略卡牌、Roguelike和杀戮尖塔类游戏的玩家，快来开启你的冒险之旅！
```

### 示例2：动作Roguelike

**简短描述**（70字）：
```
《地牢猎人》是一款动作Roguelike游戏。快节奏战斗、随机武器、挑战boss！
```

**完整描述**（250字）：
```
《地牢猎人》是一款快节奏的动作Roguelike游戏。

在随机生成的地牢中，使用各种武器和技能击败敌人。每次死亡都是新的开始，每次冒险都有新的收获。

游戏特色：
- 流畅的动作战斗系统
- 超过50种武器和技能
- 随机生成的地牢关卡
- 多个可选角色
- 挑战性boss战

适合喜欢动作游戏和Roguelike的玩家！
```

## 写作要点

1. **开头吸引人**：第一句话要让玩家知道游戏是什么
2. **突出特色**：列出3-5个明显区别于其他游戏的特点
3. **简洁明了**：避免冗长复杂的句子
4. **目标玩家**：明确告诉玩家这是给他们玩的游戏
5. **行动号召**：结尾邀请玩家体验

## 常用词汇

### 游戏类型
- 卡牌游戏 / 策略卡牌
- Roguelike / Roguelite
- 动作游戏
- 角色扮演 / RPG
- 回合制

### 游戏特色
- 丰富的...系统
- 随机生成...
- 策略深度
- 上手简单
- 多种流派
- 精美画面

### 目标玩家
- 适合喜欢...的玩家
- 推荐...爱好者体验
- ...玩家必玩
